Camp Hare WebApp

With a group I created a Text based horror game for NJIT2022 Hackathon.

Inspiration
We wanted to make something fun and something we've never done, and something we would have fun creating.
What it does
It is a website that has a text-based horror game that gives users options to pick.
How we built it
Wix API,JavaScript,HTML, CSS. As well as art made in ProCreate.
Challenges we ran into
We used languages and tools we have never used before and wanted to make this hackathon a fun learning experience. We were not able to make as much art as we would have wanted, and the struggle of learning how to code with JavaScript was challenging.
Accomplishments that we're proud of
We were able to complete a project with tools we never used before and make a functioning project. neither of us have made a website prior to this event
What we learned
Wix API,JavaScript,HTML,CSS
What's next for Camp Hare
We can update it with more art and create more endings with more efficient code. we plan on getting our own domain and hosting a website on a server we make.
